<h2>📌 Objetivo do Projeto</h2>

<p>
Este projeto tem como objetivo realizar uma <b>Análise Exploratória de Dados (EDA)</b> da
<b>Cota para Exercício da Atividade Parlamentar dos Senadores (CEAPS)</b>.
</p>

<p>
A proposta é praticar técnicas de <b>Data Wrangling</b>, incluindo:
</p>

<ul>
<li>Limpeza e tratamento de dados</li>
<li>Padronização de informações</li>
<li>Correção de inconsistências</li>
<li>Análise exploratória de dados</li>
</ul>

<p>
Além disso, o projeto busca identificar <b>padrões e insights</b> sobre os gastos parlamentares ao longo dos anos.
</p>

<hr>

<h2>🗂 Fonte dos Dados</h2>

<p>
Os dados utilizados neste estudo são disponibilizados publicamente pelo
<b>Senado Federal</b> através do portal de <b>Dados Abertos</b>.
</p>

<p><b>Base utilizada:</b></p>

<ul>
<li>CEAPS - Cota para Exercício da Atividade Parlamentar dos Senadores</li>
</ul>

<p>Esses dados incluem informações como:</p>

<ul>
<li>Ano e mês do gasto</li>
<li>Nome do senador</li>
<li>Tipo de despesa</li>
<li>Fornecedor</li>
<li>Valor reembolsado</li>
<li>Data do documento</li>
</ul>
